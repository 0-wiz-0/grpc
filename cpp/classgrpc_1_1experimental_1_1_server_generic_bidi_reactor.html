Detailed Description

ServerGenericBidiReactor is the reactor class for bidi streaming RPCs invoked on a CallbackGenericService.

The API difference relative to ServerBidiReactor is that the argument to OnStarted is a GenericServerContext rather than a ServerContext. All other reaction and operation initiation APIs are the same as ServerBidiReactor.
