Detailed Description

template<class W>
class grpc::ClientWriter< W >

Synchronous (blocking) client-side API for doing client-streaming RPCs, where the outgoing message stream coming from the client has messages of type W.

Block to create a stream (i.e.

send request headers and other initial metadata to the server). Note that context will be used to fill in custom initial metadata. response will be filled in with the single expected response message from the server upon a successful call to the Finish method of this instance.
